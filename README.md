\# Reels Auto Clipper



Automated tool to convert long videos into ready-to-post

\*\*TikTok / Instagram Reels / YouTube Shorts\*\*.



\## Features

\- Automatic 30-second clips

\- 9:16 vertical format (blur background, no cropping)

\- Speech-to-text subtitles

\- Clean subtitle styling

\- Local processing (no subscriptions)



\## Requirements

\- Python 3.11

\- FFmpeg installed and added to PATH

1\. Put a video in the `input/` folder

3\. Get results in `output/`
